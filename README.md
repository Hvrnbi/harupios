# HarupiOS

Welcome to the repo of my OS-looking portfolio !
It's a simple website where you can learn a bit more about me. (Really a bit, I don't talk so much about me, but you can play Pong if you want)
This project was started during the [Stardance challenge](https://stardance.hackclub.com/) of the [HackClub](https://hackclub.com/).

You can visit this website on https://harupios.vercel.app/.

### Features

- Your local date and time
- A dock to open the different windows
- Draggable and closable windows
- A welcome window, a book list window and an about me window
- A playable Pong game accessible by clicking 8 times on the "HarupiOS" title
